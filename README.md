# StudentAttendanceManagementSystem

## :bulb: Brief Introduction

SAMS 是一个高效、易用的学生出勤管理系统，旨在简化出勤记录与分析过程。优化日常的出勤管理任务，提升教师工作效率，同时为学校管理层提供精准的数据报表，促进学生出勤率的提升及教学资源的有效分配。使用Qt 5.14.1。
